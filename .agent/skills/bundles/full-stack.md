## How to Use

Reference these skills when:
- **Full-stack architecture** → Use `senior-fullstack`
- **Frontend development** → Use `frontend-developer` + `frontend-dev-guidelines`
- **Backend development** → Use `backend-dev-guidelines`
- **API design** → Use `api-patterns`
- **Database** → Use `database-design`
- **Next.js routing** → Use `nextjs-app-router-patterns`
- **Auth** → Use `nextjs-supabase-auth`
- **Data fetching** → Use `tanstack-query-expert` + `react-state-management`
- **AI features** → Use `vercel-ai-sdk-expert`
- **Payments** → Use `stripe-integration`
- **Deployment** → Use `vercel-deployment`
